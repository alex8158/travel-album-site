# 实现计划：旅行相册展示网站

## 概述

基于前后端分离架构（React + TypeScript 前端，Node.js + Express + TypeScript 后端，SQLite 数据库），按模块逐步实现上传管理、文件分类、图片去重、质量选择和站点生成功能，最终整合为完整的旅行相册展示网站。

  - [~] 5.1 实现感知哈希计算与去重逻辑

    - 将分组结果写入 duplicate_groups 和 media_items 表
